# 날짜 활성화/비활성화

선택할 수 없도록 특정 날짜를 비활성화해야 할 때가 있습니다.

<Sandbox example="date-management-disable-dates" />

때로는 비활성 날짜 목록을 나열하는 것보다, 모든 날짜를 비활성화한 다음 특정 날짜만 활성화하는 편이 더 쉬울 수 있습니다.
