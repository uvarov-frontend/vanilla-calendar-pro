# 날짜 클릭 처리

캘린더와의 사용자 상호작용을 위해 여러 액션이 제공되며, 그중 하나가 `onClickDate()`입니다. 이 액션을 사용하면 사용자가 캘린더의 특정 날짜를 클릭했을 때를 추적할 수 있습니다.
